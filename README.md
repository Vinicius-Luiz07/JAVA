Repositório com diversos **projetos simples em Java** focados em **aprendizado**, prática de **Orientação a Objetos** e lógica de programação.

---

## Sobre o Repositório

Este repositório contém pequenos projetos em Java desenvolvidos para consolidar conceitos fundamentais da linguagem, como:

- Orientação a Objetos (Herança, Encapsulamento, Polimorfismo)
- Manipulação de dados
- Algoritmos básicos
- Uso de classes, construtores, getters e setters
- Estruturas de repetição e condicionais

**Objetivo:** Facilitar o aprendizado e servir como portfólio inicial de Java.
